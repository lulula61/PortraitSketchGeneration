# edges2portrait

Edges to portrait image-to-image translation using MUNIT

Inspired from [Mario Klingemann's work](https://twitter.com/quasimondo/status/826065030944870400)

![edges2portrait](./result/498_2000_100.png)

## Dependency
- Python 3
- scikit-image
- numpy
- matplotlib
- keras
